hexo安装&部署github

搭建hexo博客并部署到Github上。
官网地址：https://hexo.io/
前提是已经安装了node，官网：https://nodejs.org/en/

npm install hexo-cli –g //它会在node的环境变量PATH下安装全局模块。



服务启动完毕，打开链接：http://localhost:4000/ ,博客页面。

官方默认的主题是landscape，如果需要的话可以在这里找到更多主题https://hexo.io/themes/ 。
下一步，将hexo部署到Github上。
修改blog目录下_config.yml文件,

安装部署模块，npm install hexo-deployer-git –save //–save参数会把hexo-deployer-git的模块名和版本号自动添加到blog目录下package.json中dependencies部分。

之前hexo可以使用https部署，需要输入用户名密码，但是现在使用相同的方法一直报错。


现在改为ssh提交，右键，打开Git Bash Here

之后会在 C:\Users\H.ssh 目录下生成id_rsa、id_rsa.pub两个文件，分别对应私钥和公钥。


将公钥idrsa.pub复制到该项目的Settings->Deploy keys->Add deploy key中，勾选写权限。
![](https://img.alicdn.com/imgextra/i4/1128418138/TB2WXSBfFXXXXatXXXXXXXXXXXX!!1128418138.png)
测试：

测试通过。

将_config.yml中repo的地址，修改为项目的SSH地址。

deploy:
  type: git
  repo: git@github.com:kimown/kimown.github.io.git
  branch: master

输入 hexo deploy,

打开 kimown.github.io,至此，部署完毕。
